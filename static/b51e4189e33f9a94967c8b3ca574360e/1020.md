# 문제

자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를<br> 만들어 주세요.
예를 들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

##### 제한 조건

1. N의 범위 : 100,000,000 이하의 자연수

##### 문제 풀이

```
function solution(n) {
  let answer = n.toString()
                .split('')
                .map(el => parseInt(el))
                .reduce((prev, current) => prev + current );

  return answer;
}
```

##### 블록커

각각의 자릿수를 더하는 방법이 관건이었다.

##### 끄적

```
function solution(n) {
  let result = 0;
  let str = n.toString()
              .split('')
              .map(el => result += parseInt(el));

  return result;
}
```

이전에는 별도의 변수를 생성하여 활용한 반면에<br>
이번에는 reduce Method를 활용해보았다.

reduce는 map과 동일하게 함수를 인자로 사용한다.
<br>(인자로 사용되는 함수 === 콜백함수)

map을 사용할 때는 콜백함수에 한 가지 이상의 인자를 입력해야 한다.
<br>바로 currentValue, 처리할 요소의 값을 가리킨다.

예를 들어

```
const array = [1, 2, 3];

array.map((currentValue, index) => {
  console.log(currentValue) // (1, 2, 3)
  console.log(index) // (0, 1, 2)
})
```

reduce를 사용할 때는 콜백함수에 2가지 이상의 인자를 입력해야 한다.
<br>accumulator, currentValue인데
<br>accumulator는 initialValue(첫 번째 값) 혹은 콜백함수의 반환값을 가리킨다.
currentValue는 map과 동일하게 처리할 요소의 값을 가리킨다.

예를 들어

```
const array = [1, 2, 3];

array.reduce((accumulator, currentValue) => {
  console.log(accumulator, currentValue);
  return accumulator + currentValue;
})

// 1 (=accumulator), 3 (=currentValue)
// 3 (=accumulator), 3 (=currentValue)
// 결과적으로 반환되는 값은 6;
```
