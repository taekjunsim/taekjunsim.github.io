**기록으로 남기고 싶은 아티클을 요약 및 정리하여 공유하는 공간입니다.**

[React밖에 모르는 당신에게. GatsbyJS한 잔](https://blog.banksalad.com/tech/build-a-website-with-gatsby/)<br>

- **Gatsby의 장점과 SEO를 적용하는 데에 도움이 되는 plugin에 대한 자료**

- 장점

  - React 기반이다. (Jekyll은 루비 기반)
  - 손쉽게 SEO를 적용할 수 있다.

- 플러그인
  - gatsby-plugin-react-helmet
    - meta tag 변경
  - gatsby-plugin-robots-txt
    - robot.txt 자동 생성
  - gatsby-plugin-sitemap
    - 빌드 시 sitemap 자동 생성
  - gatsby-plugin-canonical-urls
    - canonical 이슈 해결
